ICMP PING

ICMP > Internet Control Protocol, é um protocolo integrante do Protocolo IP utilizado para fornecer relatórios de erros à fonte original

PING > É uma ferramenta que usa o protocolo ICMP para testar a conectividade entre nós. É um comando disponível praticamente em todos os sistemas operacionais que consiste no envio de pacotes para o equipamento de destino e na "escuta" das respostas.
(Disponibilidade) Princípio que visa garantir que um recurso e/ou informação esteja disponível

Biblioteca os > Este módulo fornece uma maneira simples de usar funcionalidades que são dependentes de sistema operacional.

Biblioteca time > Este módulo fornece várias funçoes realcionadas ao tempo, como por exemplo tempo de execução
